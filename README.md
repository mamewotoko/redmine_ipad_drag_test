redmine_ipad_drag_test
======================

Prepare
-------
* install dependencies
  * Mac

```
brew install selenium-server-standalone chromedriver
```

Run
---
* simple test

```
sh run.sh test.py
```

* parametrized test

```
sh run.sh param_test.py
```

List iOS Simulator
------------------

```
xcodebuild -showsdks
```
